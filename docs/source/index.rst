Installation
------------

To install daniel_sudoku, simply run:

.. code-block:: bash

    pip install daniel_sudoku

Usage
-----

After installation, you can use the package by importing it in your project:

.. code-block:: python

    import daniel_sudoku

Or, you can run the package as a script:

.. code-block:: bash

    python -m daniel_sudoku
